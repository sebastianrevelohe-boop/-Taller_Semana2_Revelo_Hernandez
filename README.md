# -Taller_Semana2_Revelo_Hernandez
README.md Proyecto: 
ejercicios sobre el ciclo for en Java. Contenido: -
`Ejercicio 1`: Imprime números del 1 al 20, pares del 1 al 50 y conteo descendente del 20 al 1 usando `for`. -
`Ejercicio 2`: Crea un arreglo de 10 enteros, calcula suma, promedio y número mayor usando `for`. - 
`Ejercicio 3`: Imprime un patrón numérico incremental usando `for` anidado. - 
`Ejercicio 4`: Crea una matriz 3x3, la llena con 1..9 y la muestra en formato tabla. - 
`Ejercicio 5`: Crea una matriz 4x4, la muestra, calcula suma por filas y suma total usando `for` anidados. Cómo ejecutar: - Con Maven: `mvn compile exec:java -Dexec.mainClass="org.example.Main"` - O ejecutar la clase `org.example.Main` desde IntelliJ IDEA
